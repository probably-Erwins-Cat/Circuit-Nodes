# DC Power Supply End Node

This puzzle piece provides two connectors for a DC (direct current) power source: one for the positive pole and one for the negative pole (ground). The positive connector is red and the negative is black, following standard electrical convention. In standard current flow notation, the technical current flows from positive to negative.
